Trouble Shooting

Files or Database Problems

The problemPossible solution
The log-file is corrupt or too big You may delete manually the log-file (see his location at log-file menu title) using the OS file-manager. This application will then create a new one.
The database is corrupt or old You may delete it manually using the OS file-manager (the DB location, is the same as the log-file ones, see at the log-file menu title). This application will then create a new one.

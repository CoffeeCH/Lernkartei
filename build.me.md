# ======================================== #
# Build Informationen für FX Applikationen #
# ======================================== #

1. copy (no move) alle .css from **views.styles** to **views** folder
2. open file **build.fxbuild**
3. fill out the fields with a '*' and click on link "ant build and run"  
4. select the JAR file ind **dist** folder
5. CM->Export->**JAR executabe** to a target folder
6. delete all .css in **views** (undo step 1)
